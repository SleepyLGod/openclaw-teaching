# Sample Student Answer

OpenClaw is more than a model call because it can answer using course files and the current session. In my run, the direct LLM call replied to a short prompt, while the agent task used workspace files such as COURSE_BRIEF.md and FAQ.md. The CLI inspection step showed that the agent can ask the runtime to inspect the workspace instead of pretending it already knows every file.
