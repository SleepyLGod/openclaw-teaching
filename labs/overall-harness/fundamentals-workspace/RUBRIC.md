# Rubric

A strong student answer should:

- Distinguish a direct LLM call from an OpenClaw agent task.
- Name at least two workspace files used by the agent.
- Explain one tool-mediated action in plain language.
- Separate session state from durable memory notes.
- Identify one visible output that helps explain what the runtime did.

A weak answer usually:

- Treats the model as if it directly owns files or tools.
- Says the agent remembered something without identifying whether it came from session history, workspace files, or memory notes.
- Reports final text only, without explaining what was observed.
