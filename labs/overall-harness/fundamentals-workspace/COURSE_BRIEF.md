# Course Brief

This workspace is for an AI Agent and LLM Serving course.

The students have already learned basic agent ideas, including tool calling, RAG, memory, and planning. This lab is not teaching those ideas from scratch. It uses OpenClaw to show how those ideas appear inside a real agent runtime.

The lab package should help students observe three differences:

1. A model call only answers from the input it receives.
2. An OpenClaw agent task can use workspace files, tools, session history, and memory notes.
3. Harness engineering is the practice of shaping the system around the model so agent behavior is easier to guide, observe, and improve.

The teaching style should be concrete. Prefer short answers, named files, and observable command output over abstract claims.
