# Lab Release Checklist

Before publishing this lab, check that:

- The notebook installs OpenClaw inside the runtime.
- The notebook supports at least one API model route and one local-model route.
- API keys are never printed or committed.
- The workspace files are small enough for students to inspect.
- The first agent task is understandable before students learn the term harness.
- The tool example uses a real OpenClaw tool surface, not fake agent code.
- The final task asks the agent to cite which files or tools informed its answer.
- Slow local-model behavior is explained as a runtime capacity observation, not as a broken assignment.
